================================================================================
                            QUICK START GUIDE
================================================================================

1. INSTALLATION
   
   cd Zeitungsjunge
   python -m pip install -r requirements.txt

2. FIRST RUN

   python -m src.main

   This generates a report: Extrablatt/news_today_YYYYMMDD_HHMMSS.md

3. ADVANCED USAGE

   # Last 7 days, US sources
   python -m src.main --range 7d --region us

   # Last 30 days, German sources, fresh data
   python -m src.main --range 30d --region de --no-cache

   # See all options
   python -m src.main --help
